What is Kryptan?

A secure password storage application, for both Windows and Linux. It uses a text-based interface and works great on linux over SSH, as well as a standalone Windows application.

List of features:

  • All passwords are stored using industrial grade encryption. (3DES)
  • Retrieval of passwords needs a decryption key. (MASTER-KEY)
  • Local standalone application. (No internet access is neccesary)
  • Multi platform. Works on both Windows and Linux.
  • Password file that's platform independent (radable by both Windows and Linux versions).
  • Interface designed to minimze so called shoulder surfing
  • OpenSource

How do I compile the source-code?

You need to download and install the following prerequisites:

Crypto++
Crypto++, see http://www.cryptopp.com/wiki/Compiling for installation instructions for your platform. download it here: http://www.cryptopp.com/#download
(Linux only) Ncurses
It should probably already be installed. Please consult your distributions documentatian regarding installation instructions.
(Windows only) Pdcurses win32a
Compilation can be a bit tricky, but read the instructions on the download page: http://www.projectpluto.com/win32a.htm#download

Linux instructions

Make sure you have the gcc build suite installed and operational, and the above requirements installed where gcc can find them. After that just cd into the build/ directory an type make on the console. If everything went smoothly the binary should be located in the dist/ directory

Windows instructions (Visual studio 2010)

Open the file kryptan.sln with Visual studio. Open the project properties window and browse to "Configuration Properies -> C/C++ -> General -> Additional Include Directories" and change the value to where you have installed the the pdcurses and cryptopp directories. Next browse to "Configuration Properies -> Linker -> General -> Additional Library Directories" and change the value to include both the pdcurses/win32a and cryptopp/Win32/Output/Release directories.

Close the project property window and build the project, if everything went smoothly the binary should be located in the Release directory.

How do I use Kryptan?

Hopefully the program should be rather self explanatory, but here is some useful information:

Copy paste (Windows)

Copy/paste works a little bit differently than on normal windows applications.

To paste in a new password or uesrname, simply press the button "paste" in the menu above the main screen.

To copy a word, all you need is to select it. No CTRL+C is neccessary! To select a word drag the mouse pointer over it, alternatively you can double click on the text you want to select.

Anti keylogging

To start Kryptan in this special mode, you need to specify the command line parameter "--use-anti-keylogging".

For windows you need to create a shortcut t the application and add the switch in the properties window, like so.

Usage: In the upper left corener of the screen there are 2 rows (actually 4 since the window is to small to display the whole range of characters on one line) of the alfabet. One is in its correct order and the other is scrambled. If you want to input the password abc, you begin with the letter 'a' and press the key that is displayed above it (maybe captial 'X')

Repeat for every character of the password.
