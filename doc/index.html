What is Kryptan?

A secure password storage application, for both Windows and Linux. It uses a text-based interface and works great on linux over SSH, as well as a standalone Windows desktop application.

List of features:

  • All passwords are stored using industrial grade encryption. (256-bit AES)
  • Retrieval of passwords needs a decryption key. (MASTER-KEY)
  • Local standalone application. (No internet access is neccesary)
  • Multi platform. Works on both Windows and Linux.
  • Password file that's platform independent (radable by both Windows and Linux versions).
  • Interface designed to minimze so called shoulder surfing
  • OpenSource

How do I compile the source-code?

You need to download and install the following prerequisites:

Crypto++
Crypto++, see http://www.cryptopp.com/wiki/Compiling for installation instructions for your platform. download it here: http://www.cryptopp.com/#download
(Linux only) Ncurses
It should probably already be installed. Please consult your distributions documentatian regarding installation instructions.
(Windows only) Pdcurses win32a
Compilation can be a bit tricky, but read the instructions on the download page: http://www.projectpluto.com/win32a.htm#download
(all platforms) Boost
Please download and compile boost for your platform and add it to the include paths

Linux instructions

Make sure you have the gcc build suite installed and operational, and the above requirements installed where gcc can find them. After that just cd into the build/ directory an type make on the console. If everything went smoothly the binary should be located in the dist/ directory

Windows instructions (Visual studio 2013)

Open the file kryptan.sln with Visual studio. Open the project properties window and browse to "Configuration Properies -> C/C++ -> General -> Additional Include Directories" and change the value to where you have installed the the pdcurses, cryptopp and boost directories. Next browse to "Configuration Properies -> Linker -> General -> Additional Library Directories" and change the value to include both the pdcurses/win32a, cryptopp/Win32/Output/Release and boost/stage directories.

Close the project property window and build the project, if everything went smoothly the binary should be located in the Release directory.

How do I use Kryptan?

Hopefully the program should be rather self explanatory, but here is some useful information:

Copy paste (Windows)

Copy/paste works a little bit differently than on normal windows applications.

To paste in a new password or uesrname, simply press the button "paste" in the menu above the main screen.

To copy a word, all you need is to select it. No CTRL+C is neccessary! To select a word drag the mouse pointer over it, alternatively you can double click on the text you want to select.

Change Log

2014-04-27 - VERSION 3.1.1
  • New encryption engine, using serpent encryption and much safer encryption key derivation.
  • It is now possible to syncronize your password file to Android devices running Kryptan for Android!
2013-08-25 - VERSION 3.0
  • Completely rewritten both core and userinterface in order to make it more structured and more usable.
  • Nested categories have now been replaced by a flat list of labels, where a password can be associated by several labels.
  • New filter function where the list of passwords that matches what you type is updated in realtime.
2012-07-05 - VERSION 2.1
  • Updated: Stronger encryption! The new version replaces the old 3DES (DES-EDE2) with the much more secure AES 256-bit encryptionmethod!
  • Fixed: "Generate password" now outputs a key with correct keylength!
  • Fixed: Compiler warning messages are now handled properly!
2012-06-04 - VERSION 2.0
  • Major update! Completely rewritten code base!
  • Added capability to organize passwords into categories (or folders).
  • Updated passwordfile structure, that is incompatible with the old. Please move your passwords manualy to the new version.
2012-05-09 - VERSION 1.1.1
  • Removed the 1.1 fix, instead updated the pdcurses lib to a newer version that included its own (better) fix.
  • Keyboardlayout and latin-1 character encoding should now work correctly.
2012-05-01 - VERSION 1.1
  • Fixed keyboard layout on swedish windows machines.
  • Fixed characters on swedish windows machines.
2012-03-02 - VERSION 1.0
